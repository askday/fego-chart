## mobile

### 使用示例

~~~

	<CandleStickSeries riseColor={Const.colorRed} fallColor={Const.colorGreen} />
								
~~~

### Props

| 属性 | 说明 | 类型 | isRequired| 默认值 |
|:-- | :-- | :-- |:-- |:-- |
| riseColor | 上涨颜色 | string |否| red |
| fallColor | 下跌颜色 | string | 否 | green |
| isHollow | 上涨是否为空心 | bool | 否 | false |
